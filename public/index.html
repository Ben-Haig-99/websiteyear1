What do I study?

I study computer science and I am currently in the first year which is a more generic set of core modules which almost all similar courses are studying, these are the basics which can help you decide or confirm which path of computer science you would like to go down.

The 4 core modules which are studied in the first year are:

  1. Digital Technologies
  2. Networking Concepts and Cyber Security
  3. Software Development and Application Modelling
  4. Web Development and Operating Systems

Each of these modules is quite self explanatory what they're about and they are all quite unique from eachother meaning for the second year you're more likely to focus down 1 of these modules for the job you're looking at in the future.

My future plans

The module I like the most from the first year is Software Development and Application Modelling which helped me decide that in the second year of university I would like to take the software related modules for my 2 optional modules.

I would like to be a software engineer once finished with university as I find this the most similar to the areas of computer science that I enjoy and would be interested in persuing it further to know more about the subject such as more languages.
