CSS

CSS(Cascading Style Sheets) are used for formatting HTML pages such as changing background colour, font or sections to the page. It is done on a seperate page to the HTML and linked in the header of the HTML. You can also use CSS to edit how the page will look on a desktop and a mobile in order to make it more available to everyone from everywhere.

ECMAscript

ECMAscript is a programming language used for web development. It is a client-side language meaning the source code is processed by the client's web browser rather than on the web server. ECMAscript can be embedded into the HTML file or done in a seperate js file which can be linked in the header the same as the css does.
